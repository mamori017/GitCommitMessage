## Overview

コンテキストメニューにGitコミットメッセージを表示する

## Build

- .NET Framework 4.6.1

## Usage

- コンテキストメニューを選択するとコミットメッセージのテンプレートがコピーされる

- コンテキストメニュー"Close"でアプリケーションを終了する
